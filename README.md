# 사신수 서바이벌 (Four Guardians Survival)

전설의 사신수 힘으로 끝없이 몰려오는 적들을 물리치는 **2D 서바이벌 액션 게임**입니다.  
뱀파이어 서바이버류의 로그라이크 성장 시스템과 한국 전통 사신수(四神獸) 테마를 결합했습니다.

## 게임 소개

플레이어는 맵을 자유롭게 이동하며 자동으로 발동되는 무기와 버프로 적을 처치합니다.  
경험치를 모아 레벨업하면 새로운 무기·버프를 선택할 수 있으며, 특정 조합을 맞추면 **사신수 완성 무기**를 해금할 수 있습니다.

### 조작법

| 키 | 동작 |
|---|---|
| `W` / `↑` | 위로 이동 |
| `S` / `↓` | 아래로 이동 |
| `A` / `←` | 왼쪽으로 이동 |
| `D` / `→` | 오른쪽으로 이동 |

### 게임 흐름

1. **시작 화면** — `시작` 버튼을 눌러 게임을 시작합니다.
2. **플레이** — 적을 처치하고 경험치 구슬을 수집합니다.
3. **레벨업** — 3가지 업그레이드 중 하나를 선택합니다.
4. **게임 오버** — 체력이 0이 되면 생존 시간과 처치 수를 확인하고 다시 시작할 수 있습니다.

## 무기 & 성장 시스템

### 기본 무기 (4종)

게임 시작 시 아래 무기 중 하나가 무작위로 지급됩니다.

| 무기 | 설명 |
|---|---|
| **뇌명의 창** | 바라보는 방향으로 긴 창을 휘두릅니다. |
| **세계수의 나뭇가지** | 지나간 자리에 독 지대를 생성합니다. |
| **공허한 화령의 그릇** | 주위를 도는 4개의 칼날을 소환합니다. |
| **만파식적** | 주기적으로 적을 밀쳐내는 파동을 방출합니다. |

### 버프 (4종)

| 버프 | 설명 |
|---|---|
| **하늘의 파편** | 공격 시 20% 확률로 주변 적에게 체인 번개 |
| **맥동하는 암석** | 3초마다 주변 적 50% 둔화 (1.5초) |
| **영원의 불꽃** | 적중 시 무한 중첩 화상 부여 |
| **태풍의 눈** | 이동 속도 40% 증가 |

### 사신수 완성 무기

기본 무기와 대응 버프를 모두 보유하면 레벨업 시 완성 무기를 해금할 수 있습니다.

| 완성 무기 | 조합 | 효과 |
|---|---|---|
| **청룡** | 뇌명의 창 + 하늘의 파편 | 무작위 번개 낙하, 적 기절 및 체인 |
| **현무** | 세계수의 나뭇가지 + 맥동하는 암석 | 어둠의 소용돌이 생성, 적 기절 |
| **주작** | 공허한 화령의 그릇 + 영원의 불꽃 | 체력이 가장 높은 적에게 불타는 단검 발사 |
| **백호** | 만파식적 + 태풍의 눈 | 강력한 돌풍, 이동속도 대폭 증가 |

### 궁극 무기

4종의 사신수 완성 무기를 모두 해금하면 **황룡**을 획득할 수 있습니다.  
황룡은 모든 사신수의 힘을 각성시키는 궁극의 해방 무기입니다.

### 적 종류

- **기본 적** — 표준 속도와 체력
- **장갑 적** — 높은 체력
- **정예 적** — 강력한 공격력

시간이 지날수록 적 스폰 속도가 빨라집니다.

## 시작하기

### 사전 요구사항

- [Node.js](https://nodejs.org/) 20 이상 (권장: Node.js 24)
- [pnpm](https://pnpm.io/) 9 이상

### 설치

```bash
git clone https://github.com/mabro03/Four-Guardians-Survival.git
cd Four-Guardians-Survival
pnpm install
```

### 개발 서버 실행

```bash
pnpm --filter @workspace/sashinsu-survival run dev
```

브라우저에서 `http://localhost:5173` 으로 접속합니다.

### 빌드

```bash
# 게임 클라이언트만 빌드
pnpm --filter @workspace/sashinsu-survival run build

# 전체 워크스페이스 타입체크 + 빌드
pnpm run build
```

빌드 결과물은 `artifacts/sashinsu-survival/dist/public` 에 생성됩니다.

### 프로덕션 미리보기

```bash
pnpm --filter @workspace/sashinsu-survival run serve
```

## 프로젝트 구조

```
Four-Guardians-Survival/
├── artifacts/
│   └── sashinsu-survival/     # 게임 클라이언트 (메인)
│       └── src/
│           ├── game/          # 게임 엔진, 무기, 적, 버프 로직
│           ├── components/    # HUD, 업그레이드 화면, UI
│           └── pages/         # 게임 페이지
├── lib/                       # 공유 라이브러리 (API 클라이언트, DB 등)
├── artifacts/api-server/      # Express API 서버 (선택)
└── package.json               # 워크스페이스 루트
```

### 핵심 게임 파일

| 파일 | 역할 |
|---|---|
| `src/game/engine.ts` | 게임 루프, 레벨업, 카메라, 상태 관리 |
| `src/game/weapons.ts` | 무기 정의 및 공격 로직 |
| `src/game/enemies.ts` | 적 스폰 및 AI |
| `src/game/buffs.ts` | 버프 정의 및 효과 |
| `src/pages/game.tsx` | Canvas 렌더링 및 게임 화면 |
